Using the Command Line (Git Bash, Terminal, or CMD)
This is the most common way to download a full copy of the project data, including all versions of every file.
Find the URL: On GitHub, go to the main page of the repository you want to clone.
Copy the Link: Click the Code button above the file list and copy the provided URL under the HTTPS or SSH tab.
Open Terminal: Open your terminal (Mac/Linux) or Git Bash (Windows).
Navigate: Use the cd command to move to the folder where you want the project to live.
Run the Command: Type the following and press Enter:
git clone <paste-the-url-here>
git clone https://github.com
